Adaptive Nerf Targeting System

1. Abstract

Our project is an adaptive targeting system that uses pressure sensor feedback to automatically realign a motor-controlled platform that holds a Nerf gun toward the center of a target. It consists of two communicating pieces: a Nerf gun stand that moves up, down, left, and right, and a pressure-sensor-covered target that sends feedback for alignment corrections.

2. Motivation

The problem we are solving is the aiming of a projectile. Our project controls a Nerf gun with a 2-axis motor system informed by feedback from a peripheral target device, aiming for more accurate firing.

4. Conclusion

What we learned: We gained hands-on embedded systems experience, especially with RF modules, motor control, and precision mechanisms. We also learned the value of careful planning before construction.

What went well: The full system loop — fire, hit, adjust, fire again — worked reliably. We achieved fast RF communication, effective up/down movement, and a consistent trigger mechanism.

What we're proud of: Integrating independent modules into a functioning autonomous system that consistently realigned itself toward the target.

Changes we made: We redesigned the motor stand to separate vertical and horizontal movement for stability and precision. We replaced earlier design ideas with more effective nut-and-bolt linear motion.

Unexpected challenges: Inconsistent Nerf bullet trajectories added randomness that was difficult to fully compensate for despite precise motor adjustments.

Next steps: Future improvements could include building a more accurate firing mechanism and adding a vision-based aiming system using cameras and computer vision.
